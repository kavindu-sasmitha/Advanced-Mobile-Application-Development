# VibeCast ЁЯУ║

VibeCast is a **live streaming and private chat mobile app** built with Flutter. Users sign in with Google, set up a profile, go live to broadcast video to other users, and can request private one-on-one chats with people they meet through live streams.

---

## тЬи Features

- **Google Sign-In authentication** via Firebase Auth
- **Profile setup** тАФ name, age, country, district, birthday, gender, and profile picture
- **Live streaming** тАФ go live with camera/mic, viewers can watch in real time
- **Realtime comments & likes** on live streams
- **Live viewer count**
- **Private chat requests** тАФ a viewer can request to chat privately with a streamer; the streamer can accept or reject
- **Realtime 1-on-1 chat** once a request is accepted, showing the other user's profile name and photo

## ЁЯЪА Getting Started

### Prerequisites
- Flutter SDK
- A Firebase project (Authentication, Firestore, Storage enabled)
- An Agora.io account and App ID

### Setup
```bash
flutter pub get
dart pub global activate flutterfire_cli
flutterfire configure --platforms=android
```

Add your Agora App ID in `lib/config/agora_config.dart`:
```dart
static const String appId = "YOUR_AGORA_APP_ID";
```

### Run
```bash
flutter run
```

### Build a release APK
```bash
flutter build apk --release
```
Output: `build/app/outputs/flutter-apk/app-release.apk`

## ЁЯФТ Security Notes

Firestore and Storage security rules (`firestore.rules`, `storage.rules`) enforce that:
- Only signed-in users can read/write data
- Users can only edit their own profile
- Chat requests can only be created by the sender and responded to by the recipient
- Chat messages are only accessible to the two participants

For production, Agora should be switched from App-ID-only authentication to **token-based authentication** (a backend-issued short-lived token per channel) instead of the current testing-mode setup.
